Set up the simulation with an ODE solver.

Per default Runge Kutta Cash Karp 54 solver is used. Use function set_integrator() to set a different integrator.

Parameters
[in]modelAn instance of a LCT model.
[in]t0The Start time, usually 0.
[in]dtInitial step size of integration.
